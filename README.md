## Installation

### 1. Clone the repository

```bash
git clone https://github.com/h00kmeat/anatooly.git
cd anatooly

2a. Poetry
--------------------------------
pip install -r requirements.txt
poetry install
anatooly --help
--------------------------------
poetry run anatooly --help

2b. pip+GitHub
--------------------------------
pip install git+https://github.com/h00kmeat/anatooly.git
anatooly --help
